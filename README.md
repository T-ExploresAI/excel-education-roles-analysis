# Excel Education Roles Analysis

## Overview

This project demonstrates my ability to clean, structure, and analyse a dataset of education roles using Microsoft Excel.

The dataset tracks anonymised education placements including role type, country, and duration. The goal was to transform raw structured data into a clean, dynamic, and presentation-ready summary.

---

## Skills Demonstrated

- Data cleaning and anonymisation
- Converting ranges into structured tables
- Structured references
- Dynamic formulas (COUNTIF, SUM, AVERAGE, IF, DATEDIF)
- Use of UNIQUE for category extraction
- PivotTable aggregation and layout formatting
- Basic data visualisation (bar chart)
- Summary dashboard layout design

---

## Key Features

- Automatically updating duration calculations
- Dynamic role and country counts
- PivotTable breakdown by country and role
- Clean presentation formatting for reporting

---

## Tools Used

- Microsoft Excel
- Structured Tables
- PivotTables
- Basic Charting
